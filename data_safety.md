# Google Play データセーフティ回答テンプレート — みちのわん

> Play Console の「アプリのコンテンツ」→「データ セーフティ」で入力する内容をまとめたもの。

---

## 概要回答

| 質問 | 回答 |
|------|------|
| アプリはユーザーデータを収集または共有しますか？ | **いいえ** |
| アプリはデータ収集に関するセキュリティ対策を講じていますか？ | **はい**（データはデバイス内のlocalStorageのみに保存。転送なし） |

## データの種類ごとの回答

| データの種類 | 収集するか | 共有するか | 説明 |
|-------------|-----------|-----------|------|
| 位置情報 | **しない** | — | 位置情報APIを使用しない |
| 個人情報（名前、メール等） | **しない** | — | ユーザー登録機能なし |
| 財務情報 | **しない** | — | 決済はStripe経由（将来）。カード情報をアプリが保持しない |
| 健康・フィットネス | **しない** | — | 該当機能なし |
| メッセージ | **しない** | — | メッセージ機能なし |
| 写真・動画 | **しない**（※） | — | スタンプ写真の撮影機能あり。撮影した写真はデバイス内（localStorage）にのみ保存され、外部サーバーへの送信は一切行いません |
| 音声 | **しない** | — | 音声認識は端末内処理。録音データを保存・送信しない |
| ファイル・ドキュメント | **しない** | — | バックアップJSONの書き出し・読み込みはユーザー操作のみ |
| カレンダー | **しない** | — | 該当機能なし |
| 連絡先 | **しない** | — | 該当機能なし |
| アプリのアクティビティ | **しない** | — | アクセス解析・トラッキングツールを使用しない |
| ウェブ閲覧 | **しない** | — | ブラウザ履歴等を取得しない |
| デバイスID等の識別子 | **しない** | — | 広告IDや端末IDを取得しない |

## データの暗号化

| 質問 | 回答 |
|------|------|
| 転送中のデータは暗号化されますか？ | **該当なし**（サーバーへのデータ送信なし。HTTPS配信のみ） |

## データ削除

| 質問 | 回答 |
|------|------|
| ユーザーはデータの削除を要求できますか？ | **はい**（ブラウザのサイトデータ削除で完全消去可能） |

## コンテンツレーティング（IARC）回答用

| 質問 | 回答 |
|------|------|
| 暴力的な内容がありますか？ | いいえ |
| 性的な内容がありますか？ | いいえ |
| 恐怖を煽る内容がありますか？ | いいえ |
| 薬物・アルコールに関する内容がありますか？ | いいえ |
| 差別的な内容がありますか？ | いいえ |
| ギャンブルの要素がありますか？ | いいえ |
| ユーザー間のコミュニケーション機能がありますか？ | いいえ |
| 現在地の共有機能がありますか？ | いいえ |
| デジタル商品の購入機能がありますか？ | はい（みちのわんマスター: 買い切り980円） |
| 広告を含みますか？ | いいえ（現時点では広告を含みません） |
| 対象年齢 | 全年齢（ただし子ども向けではない） |
